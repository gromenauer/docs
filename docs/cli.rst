The PyWeek CLI tool
===================

There is a ``pyweek`` command-line tool available on PyPI which makes it easier
to download and rate the entries.

Install it with:

.. code-block:: none

    pip install pyweek

To download the entries, use:

.. code-block:: none

    pyweek download <comp number>

For example:

.. code-block:: none

    pyweek download 27
